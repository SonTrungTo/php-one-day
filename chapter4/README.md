## Variables
- `\$[^0-9]\w+`
- `echo '$1'` => print `$1`
- `echo "$1"` => print `value of $1`
## Basic data types
- `int`: too large ? `float` : `int`
- `float`: note, `7E11` === `7 x 10ยนยน`
- `bool`
- Verify data type by `var_dump($x)`
## Type casting
Put
- (int)
- (float/double)
- (bool)
- (array)
- (string)
- (object)
in front of a value => convert
